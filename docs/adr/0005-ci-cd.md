# 0005 - CI/CD inicial

- Status: Proposed
- Context: Monorepo pnpm con apps Next y paquetes compartidos necesita pipelines básicos y gates antes de prod.
- Decision: GitHub Actions con workflows separados: CI (lint/typecheck/test/build placeholders) y planificación de DB; Node 22 + pnpm 9; se requerirá aprobación para migraciones y deployment.
- Consequences: Cada PR debe pasar CI; se añadirán jobs reales (lint, typecheck, tests, build) y validación de migraciones; se evita bypass de seguridad en pipelines.
